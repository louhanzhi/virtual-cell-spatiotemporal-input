# 小论文任务清单 · 虚拟细胞的时空 input

## ① 要交什么
- 体裁：**perspective（观点综述）**，不是 systematic review
- 字数：**1500–2000 字**
- 性质：**有明确立场、围绕一个论点展开**；文献是支撑论证的弹药，不是综述对象
- ⚠ 动笔前与老师确认：字数/格式是否照此、骨架可否调整、课程作业还是投稿、中文还是英文

## ② 核心论点（Thesis）
> 虚拟细胞的输入之争，本质是要不要承认细胞活在空间与时间里——而当前的数据范式恰恰把这两者都丢掉了。

写作要全程服务这一句，每一节都在为它弹辩。

## ③ 必读论文（动笔前读这 4+1 篇，按此顺序）
1. **Bunne et al. — How to Build the Virtual Cell with AI**（Cell 2024）— 定调、愿景纲领
2. **Nicheformer**（Nat Methods 2025）— "空间必须当 input"的最强证据
3. **Waddington-OT**（Schiebinger et al., Cell 2019）— 时间/轨迹奠基
4. **stVCR**（Nat Methods 2026）— 单细胞分辨率最近的时空方法
5. **ST 选型实操指南**（PMC）— 测量端约束的弹药库

补充/选读：Multi-scale AIVC review (BiB 2026)、AIVC preclinical (npj DM 2025)、CellPLM、Spateo、亚细胞 ST 基准 (Nat Commun 2025)、LivecellX、Caliban、CellProphet

## ④ 五节骨架（字数配比 + 每节要塞的料）
见 `04_drafts/outline.md`。

## ⑤ 不可替代性（被问 novelty 时的答案）
没有任何一篇把这三件事合在一篇里讲：
1. 把**空间 + 时间**作为细胞级 input 的两条正交轴
2. 用**数据需求/规格**而非"模型能力"做锚
3. 接上**测量端的分辨率-通量-破坏性三难**（决定 input 能不能拿到）
中文圈更是空白，现有都是对 Bunne 那篇的科普转译。

## ⑥ 交稿后可让人帮你做的三件事
- (a) 逐节对照论文查重，看有没有撞上别人说过的点
- (b) 补齐引用、核对数据准确性
- (c) 挑论证薄弱、可加洞见的地方
