## 1. 核心主线 / THESIS（全文只论证这一件事）

> **AI 虚拟细胞在模型上狂奔，却建立在一个结构性丢失「空间」与「时间」的数据地基上。** 主流范式——解离式单细胞测序——既把**空间组织**打散，又因测量是**破坏性**的而无法追踪**时间轨迹**；同时模态偏窄（近乎只有 RNA），而**各模态彼此不可替代**——蛋白、代谢、空间分布等各自承载着 RNA 无从代偿的信息。虚拟细胞真正需要的，是一套**显式编码时空、覆盖这些不可互替的多模态、可比可整合的数据标准（"算据标准"）**。

**一句话压缩**：虚拟细胞缺的不是模型，是**时空分辨、可整合可比**的多模态算据——我们来定义并补上它。

## 2. 原始构思

这些来自 AIVC Webinar（作 motivation 语气，不作正式 citation）：

- **单细胞数据缺时空**：郭天南——「单细胞的数据一般没有时间的信息……没有 time course……很多单细胞的数据还没有空间的信息」；「时间空间的数据实际上是挺少的」。
- **AI 侧自承缺空间/时间/蛋白模态**：高张扬——「数据主要还是从 scRNA 去做的……蛋白质模态，以及空间和时间上的信息也是缺失的」。
- **破坏性测量 = 无法成对观测（时间轴硬约束）**：高/袁均独立指出——细胞测完即死，「天生没法成对地被观测到」，监督信号只能"分布到分布"。
- **数据孤岛 + 无标准**：郭天南——「已有的数据都是单独产生的，都没有一个标准」，不同数据「不一定可以很好地串起来」。
- **跨论文不可比**：control 选择、随机切分会大幅改变结果，「同样的模型完全不可比」→ 凸显标准缺位。
- **方向背书**：高张扬——「带扰动的空间微环境……是下一个阶段细胞扰动模型要解决的问题」；主张「打破数据孤岛，训练全尺度、全模态的模型」。
- **议程契合**：八月西湖 AIVC Week 将专门讨论 **AIVC 数据标准 / QC / 如何 benchmark 不同数据**——本文主题正落在这一议程里。

## 3. 论证主线的逻辑链（文章的骨架）

1. **瓶颈在数据，不在模型。** 基础模型换个数据集就崩；上限被 input 卡住。(无论是机理模型还是data-driven的模型都缺高质量input数据, 就是要想清楚什么样标准的数据才是虚拟细胞建模所亟须的高质量数据)
2. **当前数据丢了时间。** 测序是破坏性的 → 同一细胞无法被前后成对观测 → 时间只能从分布反推，而非真实轨迹。
3. **当前数据丢了空间。** 解离把细胞从其空间组织/微环境中剥离 → 亚细胞定位、组织微环境信息消失。
4. **当前数据模态太窄。** 近乎只有转录组；而**各模态彼此不可替代**——RNA、蛋白、修饰、复合物、代谢、空间分布各自承载其他模态无法代偿的信息，只测 RNA 拼不出完整细胞
5. **当前数据是孤岛、不可比。** 各自产生、无统一标准、跨论文 benchmark 不可比 → 拼不成一个细胞。
6. **结论 = 算据标准。** 虚拟细胞需要一套**显式编码时空 × 跨模态 × 可整合可比**的数据标准；本文给出它应包含什么。
7. **我们能补。** 郭组高通量时空蛋白质组学直接对位"蛋白 + 空间 + 时间"缺口，且有条件参与定义标准（呼应八月 AIVC Week）。

## 4. 「算据标准」要论证整理的内容（= 文章主体）

把标准拆成**三根正交的轴**，各类数据落在轴的交叉上：

**轴 A · 模态（measure what）**——每一类都要论证"虚拟细胞为何需要它、现状如何"：

| 模态         | 在虚拟细胞里的角色            | 现状                   |
| ---------- | -------------------- | -------------------- |
| 基因组 / 序列   | 零件清单、底图              | 成熟                   |
| 转录组        | 表达动态（不可替代，但代偿不了下游各层） | 充足却被过度依赖             |
| **蛋白质组**   | 功能执行层，RNA 无从代偿       | **被忽视得最重 · 我们的产能优势** |
| PTM / 蛋白状态 | 活性、开关                | 稀缺                   |
| 互作组 / 复合物  | 分子机器                 | 部分                   |
| 代谢组 / 通量   | 生化状态与流               | 部分                   |
| 调控 / 信号网络  | 谁调控谁                 | 部分                   |
| 影像 / 形态    | 形态与定位                | 有                    |
| 表型 / 扰动响应  | "接地"信号（监督）           | 需配对                  |

**轴 B · 时间（temporal）**——每个模态都应有 time course / 轨迹；正视"破坏性测量无法成对观测"这一硬约束。

**轴 C · 空间（spatial）**——每个模态都应有空间分辨：亚细胞定位 → 细胞 → 组织微环境；解离范式恰恰抹掉这一维。

**贯穿 · 标准与整合（standard）**——QC、跨实验室/平台可比、跨模态可配对、统一参照系；否则数据再多仍是孤岛。

> 即：**算据标准 = 模态 × 时空分辨 × 可整合可比**。当前数据范式在每根轴上都有系统性缺失

## 5. 需要广泛搜集文献来支撑的论点清单（→ 取证路线图）

沿这些 claim 去搜集、整理文献（每条都是一段论证的弹药）：

1. **数据是瓶颈、纯 data-driven 模型泛化差** → VC/AIVC benchmark 与基础模型泛化批评。
2. **scRNA-seq 丢失空间信息** → 空间组学（转录/蛋白）综述与方法。
3. **破坏性测序无法成对观测时间** → 轨迹推断 / 谱系追踪 / Waddington-OT 类。
4. **各模态信息正交、彼此不可替代**（核心弹药：RNA 测不出蛋白、蛋白测不出代谢……单模态拼不出细胞）→ mRNA–protein 相关性、翻译/降解调控、多组学互补文献。
5. **蛋白、代谢等模态对功能与表型不可或缺** → 各模态驱动的细胞状态/疾病研究。
6. **高通量时空蛋白质组学已可行** → 领域方法进展 + 我们自己的能力/数据。
7. **空间组织/微环境改变细胞状态与扰动响应** → 微环境、空间邻域效应文献。
8. **多模态彼此提供正交信息、需整合** → 多组学整合方法与案例。
9. **数据孤岛 / 跨论文不可比 / 缺标准** → 数据标准、QC、benchmark 可比性文献。
10. **分辨率 × 通量 × 破坏性的"三难困境"** → 测量端 trade-off（bulk vs 单细胞、in vitro vs in vivo）。
11. 不局限于以上的线索, 若是思考到有合理的重要的需要补充的随时可以去调研, 以上只作为起步阶段方向

- **会议素材只作 motivation / 现场观察语气，不作正式 citation**；正式引用走 references。
