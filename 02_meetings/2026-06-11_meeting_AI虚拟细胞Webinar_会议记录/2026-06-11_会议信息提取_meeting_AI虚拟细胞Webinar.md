## 0. 这场会议对本论文为什么有用（一句话）

会议两位 speaker（高张扬讲 VC-Bench / 细胞扰动基础模型，袁新玉讲 Perturb-Diff）都是**纯扰动建模、纯 scRNA 数据驱动**的代表路线；而郭天南在问答里**反复把话题引向"时间 / 空间信息缺失"和"数据标准"**。也就是说：**会议本身是本论文 thesis 的活样本**——领域最前沿的工作恰好印证了"时空 input 被结构性低估"这一缺口，且这个缺口是由领域内的资深 PI 当场点出来的。

## 1. 直接命中 thesis 的发言（最高价值，可直接做论文弹药）

### 1.1 郭天南：单细胞数据"一般没有时间信息"，时空数据"实际上挺少"
- **原文（line 93）**：「就单细胞的数据呢，它一般是没有没有那个时间的信息……没有 time course。当然有一些空间转录组，但很多单细胞的数据还没有空间的信息……只有那些空间转录组还有空间的信息。这个可能也是后面要讨论。」
- **原文（line 233）**：「对，就时间空间的数据实际上是挺少的。」
- **对应骨架**：第 1 节（为什么时空是关键 input）、第 4 节（三难困境）。这是**领域 PI 当场承认主流单细胞数据缺时空**，可作为引文之外的"现场证据"佐证 thesis。

### 1.2 高张扬：AI 侧主流路线明确缺空间 / 时间 / 蛋白模态
- **原文（line 19）**：「数据现在大家还主要还是从 scRNA 去做的……但是它还是覆盖不了其他的一些模态……郭老师这边非常关心的蛋白质模态，以及他们在空间和时间上的信息也是缺失的。然后就是跨细胞类型、跨越疾病的状态加上 DOS 信息，特别是在不同时间下，还有组合扰动这些 OOD 的泛化能力，现在的模型都是非常弱的。」
- **对应骨架**：第 1 节、引子。**AI 研究者自承"空间和时间信息缺失"**——和本文论点完全同向。

### 1.3 高张扬（收尾问答）：带扰动的空间微环境 = "下一阶段要解决的问题"
- **原文（line 229）**：「只有一个对呃带扰动的空间微环境。我觉得这是一个……很好的能力建设方向，只是说现在单细胞的这些模型还没有把空间信息或者微环境的这些变量建模进去。就是这……是一个很好的方向，可能是下一个阶段细胞扰动模型要解决的问题。」
- **价值**：把"空间微环境作为 input/建模变量"明确定位为**下一阶段方向**——可用于第 5 节"展望"，呼应 outline 里"input 设计必须显式编码时空"。

### 1.4 破坏性测序 = 细胞天生无法成对观测（时间轴的硬约束）
两位 speaker 都独立强调了这一点，正是 outline 第 3 节"破坏性快照 vs 非破坏性活成像"的核心：
- **高张扬（line 19）**：「单个细胞它准确的这个变化是不靠谱的，因为这个细胞只要测量之后它就死掉了，所以它天生没法成对地被观测到。」
- **袁新玉（line 75）**：「测序它一般是破坏性的。一个细胞被测完之后，我们不能再对同样一个细胞施加另外的扰动并观测它之后的状态……所以监督信号是分布到分布的，而不是成对的 cell 到 cell 的。」
- **价值**：这是"为什么时间轴只能反推、为什么破坏性是硬约束"的第一手论述，可与 Waddington-OT / stVCR（003/004）互为印证。

### 1.5 in vitro vs in vivo、bulk vs 单细胞——测量端 trade-off（呼应"三难困境"）
- **bulk vs 单细胞（郭天南问，line 85；袁新玉答，line 89）**：单细胞通量高（百万-千万级）但破坏性、噪声大；bulk 信号稳、噪声小但是 average。袁提出可"先 bulk 粗粒度、再单细胞细粒度"的层级思路。
- **in vitro vs in vivo（观众问，line 209；袁答，line 213）**：RNA-seq 与 perturb-seq 测序深度、可测基因数（几万 vs 更浅 / 更窄）、信号可靠度都不同。
- **对应骨架**：第 4 节"分辨率 × 通量 × 破坏性三者不可兼得"。
  > 批注：会议讨论的是 bulk/单细胞、in vitro/in vivo、药物/基因扰动这几组 trade-off，**与 outline 第 4 节的"分辨率×通量×破坏性"不是同一组轴**，但同属"测量端硬约束"。引用时需说明这是相邻而非等同的论证，避免张冠李戴。

## 2. 间接相关：领域现状证据（佐证"纯 data-driven 范式撑不起泛化"）

这一组不直接讲时空，但支撑 outline 的底层判断"静态快照里猜不出时空，模型上限被 input 卡住"。

### 2.1 高张扬：不相信"单/少数据集 data-specific foundation model"范式
- **原文（line 19）**：「我对目前虚拟细胞领域这种纯 data-driving、特别是基于 data-specific、基于一两个数据集去训一个 foundation model 的范式，我是不相信它的……换一个数据集它基本上马上就崩掉了。」
- **主张打破数据孤岛**：「是不是可以把分子扰动、细胞状态、组织微环境，甚至更多数据有机连接，训练一个全尺度、全模态的模型。」
  > 批注：高这里说的"组织微环境 / 全尺度全模态"和本文的"空间 input"高度同向，但他的语境是**跨数据集泛化**而非"input 该含哪些时空字段"。可引用其"组织微环境"措辞，但不要把他的结论直接等同于本文论点。

### 2.2 郭天南：连数据也未必能串起来（对"多组学/多模态万能论"的反驳）
- **原文（line 31）**：「同一个疾病、同一个肺癌，不同病人细胞不一样，甚至一个病人不同肺癌细胞也可能不一样……你这个数据跟另外一个数据不一定可以很好地串起来……目前已有的数据都是单独产生的，都没有一个标准。」
- **价值**：可作为引子/第 1 节的"反方观点"——**说明仅靠堆模态/堆数据不解决问题，更凸显"input 维度本身（时空）选错了"才是症结**。

### 2.3 Benchmark / 评估不可跨论文比较（数据标准缺失）
- 高张扬：VCC 比赛三个指标相互冲突且"可被人为攻击"；随机切分会显著高估 OOD 泛化（line 19）。
- 袁新玉：control cell 的选择会"大幅影响最后结果"，导致**跨论文指标不可比**（line 189、197）。郭天南据此提出 concern：「不同论文采取不同变量测出来的结果，同样的模型完全不可比」（line 193）。
  > 批注：这条与本论文 thesis（时空 input）**不直接相关**，属于"数据/评估标准"议题。但它和下面 §3 的"8 月数据标准讨论会"强相关，对项目推进有用，故保留。

## 4. 两个报告的速记（背景，便于判断上面论点的语境）

### 报告一 · 高张扬（上海 AI Lab；西湖大学毕业，导师李子青）
- 题目：AI 虚拟细胞模型进展与评估反思。
- 自研模型：latent flow matching（by-level set encoder/decoder，解决基因排列的置换不变性），在**细胞集合层面**建模扰动前后变化；基于 BioNeMo 分布式框架，比 STATE 官方实现加速 10×+。
- **VC-Bench**（送审 Nature Machine Intelligence）：用 SC-foundation embedding 做严格切分（非随机/留一），发现随机切分**显著高估** OOD 泛化；多指标相互冲突；整合 11 个模型做头对头比较；线性模型在部分场景比复杂模型更稳。
- 三大未来重心：①打破数据孤岛、全尺度全模态；②可信 benchmark；③可解释 + 实验闭环。
  > 批注：模型名称"skill / SAGBT / Jinformer / SA Foundation / 塔后100mini / 盘后 x one / Xcell"等均为转写音译，**引用前务必核实**（疑似 scGPT、Geneformer、scFoundation、Tahoe-100M 等，但不确定）。

### 报告二 · 袁新玉（蒙特利尔大学 / Mila；导师滕建?，本科北大）
- 题目：**Perturb-Diff: Functional Diffusion for Single-Cell Perturbation Modeling**（ICML 2026 poster）。
- 核心：扰动预测是**分布到分布**问题，而非点估计；用 **Kernel Mean Embedding** 把 cell distribution 映射进 RKHS / Hilbert space，再在其上做 diffusion（MMDiT 架构，双 token stream 融合 control 与 perturb）；KME 的 L2-norm loss 可无损翻译回 cell space 的 MMD loss。
- 关键卖点：观测不到的**微环境 / 局部状态**会诱导分布级多样性，模型须显式表达响应的不确定性；用大规模 single-cell Atlas 预训练 + 扰动数据 fine-tune，在稀疏数据场景增益显著。
- 数据集：PBMC（~10M cell）、Tahoe-100M（~100M+ cell，药物扰动）、Replogle（~0.6M，CRISPRi 基因扰动）。
  > 批注：导师"滕建"、数据"PPMC / Talawangzhengmiyian / REPLUG / 塔后100million"为音译，**需核实**（疑似 Tahoe-100M、Replogle）。
  > 批注（与 thesis 的接点）：袁反复强调"**未观测的微环境会改变扰动响应**"——这正是本论文"空间微环境必须作为 input"的侧面佐证，可引为"连扰动建模社区自己都承认微环境是隐变量"。但他用 diffusion 的**不确定性**来 cover 这个隐变量，而本文主张应把它**显式编码为 input**——这是一个可以拿来对比的张力点。
