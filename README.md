# Virtual Cell · 细胞级时空数据 input（perspective 观点综述）

> **核心论点（thesis）**：虚拟细胞的输入之争，本质是要不要承认细胞活在空间与时间里——而当前的数据范式恰恰把这两者都丢掉了。

本仓库收纳这篇 perspective 写作过程中的全部材料：外部文献、会议转录、团队讨论、论文草稿、自制图表、任务/沟通记录。

- **体裁**：perspective（观点综述），非 systematic review
- **字数**：1500–2000 字
- **可见性**：private（含版权 PDF 原文）

## 命名约定

- **文献文件名**：`年份_第一作者_关键词.pdf`（如 `2024_Bunne_VirtualCell.pdf`）
- **会议与草稿**：日期前缀（如 `2026-06-15_meeting.md`、`2026-06-15_draft_v2.md`）
- **文献管理**：以 [`references.bib`](01_literature/references.bib)（BibTeX）为引用源；引用状态在 [`reading-list.md`](01_literature/reading-list.md) 追踪。
